<h3>What is HTML?</h3><br>
HTML stands for Hyper Text Markup Language.<br>
HTML is the standard markup language for creating Web pages.<br>
HTML describes the structure of a Web page.<br>
HTML consists of a series of elements.<br>
HTML elements tell the browser how to display the content.<br>
Basic structure of every website is created using HTML Language.<br>
We can create a static website by HTML only.<br>


<h3>History of HTMl</h3><br>
1. In 1993 , Tim Berners Lee developed HTML 1.0 , the first version of HTML
2. 1991 Tim Berners Lee created HyperText Markup Language but it was not officially released.
3. 1993 Tim Berners Lee created the first version of HTML 1.0 that was published and available to the public.
4. 1995 HTML 2.0 was released with a few additional features along with the existing features.
5. 1997 There was an attempt to extend HTML with HTML 3.0 , but it was replaced by the more practical HTML 3.2.
6. 1999 HTML 4.01 , which became an official standard in December
7. 1999, was the most widely used version in the early 2000s.
8. 2000 XHTML 1.0, completed in 2000, was a combination of HTML4 in XML.
9. 2012 HTML5 can be seen as an extended version of HTML 4.01, which was officially published in 2012.
